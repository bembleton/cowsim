# The Epic of CowSim

## Controls

### In-Game

- **DPAD** - Move
- **Double Tap DPAD** - Dash
- **A** - Primary Attack
- **Hold A** - Power Attack
- **B** - Use Secondary Item
- **START** - Toggle the Game Menu
- **START + UP** - Open Map Menu
- **SELECT + DPAD** - Switch to Favorite
- **SELECT + A** - Open Weapon Menu
- **SELECT + B** - Open Item Menu


### Menu

- **SELECT** - Next Menu
- **DPAD** - Move Selection
- **A** - Select/Use Item
- **Hold A + DPAD** - Set Favorite
- **B** - Drop Item
